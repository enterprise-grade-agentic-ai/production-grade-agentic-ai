# Challenge: IDE Setup

The goal of this challenge is to get your development environment ready for all the challenges ahead. This means having Python installed and an AI-assisted IDE set up and working. The demos use **Cursor**, but you're welcome to use any IDE you're comfortable with, such as **Visual Studio Code** (with GitHub Copilot or the Claude extension).

> **Cost note:** The IDEs mentioned here are free to download. Some, like Cursor, have a free tier for AI features — that's enough for this challenge.

---

## Steps

1. **Install Python** if you haven't already. Search online or ask an LLM like ChatGPT or Gemini for instructions specific to your operating system.

2. **Download and install your IDE.** The demos use [Cursor](https://cursor.com/) — download it for your OS. Alternatively, set up [Visual Studio Code](https://code.visualstudio.com/) with an AI extension of your choice.

3. **Create a new project.** Launch the IDE and open a new folder for your project files. Keeping each project in its own folder makes your workspace easier to manage.

4. **Try out the AI chat.** Open the IDE's chat or AI assistant pane and give it a prompt to generate some Python code. You can use any prompt you like, or use the exact prompt from the demos — it's available in the [README.md](README.md) file in this folder. Watch the steps the AI takes to produce the result. This gives you an early feel for how an agent acts on a goal and makes decisions along the way. Since the prompt includes running tests, it will also invoke Python to execute them — which confirms your Python install is working.

   > Note: The AI may occasionally get test cases wrong on the first try, but it will attempt to self-correct.

5. **Review the generated code** and get comfortable navigating it in your IDE.
