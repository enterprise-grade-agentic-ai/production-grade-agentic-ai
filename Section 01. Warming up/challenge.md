# Challenge: Try out a coding agent

> **Cost note:** Lovable offers a free tier that's sufficient for this challenge — no paid plan needed.

This challenge is about playing around with a coding agent that generates code and UI based on your instructions. Try out [Lovable](https://lovable.dev/) or any other code generation agent you prefer.

---

## Steps

1. **Register** with a code generation agent (e.g., [Lovable](https://lovable.dev/)). Most agents ask for an email to sign up — this is generally fine as they have privacy policies in place. If you'd prefer not to sign up, look for an alternative agent that doesn't require it.

2. **Give it a prompt** to generate some code. You can use any prompt you like, or use the exact prompt from the demos — it's in the [README.md](README.md) file in this folder. Once generation starts, watch the steps it takes to reach its goal — this is a great early glimpse at how an agent makes decisions and acts autonomously throughout a task.

3. **Check out the result.** Browse the generated UI and peek at the code if you're curious.

4. **Iterate.** If you don't like the initial look, just tell the agent to change it — for example: *"Make the UI theme lighter"*. The agent remembers the original request and your edits, and adjusts accordingly.
